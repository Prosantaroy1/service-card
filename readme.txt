== Installation ==

### From Gutenberg Editor:
1. Go to the WordPress Block/Gutenberg Editor
2. Search for **Service Card**
3. Click on the **Service Card** block to add it

### Download & Upload:
1. Download the **Service Card** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`service-card.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

### Manually:
1. Download and upload the **Service Card** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
